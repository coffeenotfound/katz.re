My Journey

2008

I discovered GameMaker

and with that my interest in logic and game design. Though I started with visual coding I later switched to the integrated programming language GML.

2009

I found Minecraft

which basically introduced me to most of what I do nowadays: Coding, modding, tinkering, pixelart, hardware logic and more.

2010

I started learning Java

to be able to make modifications for Minecraft, sparking my passion for coding. Though I've learned mutliple languages since, Java has been my go-to language because of it's beautiful nature.

2014

I got into realtime rendering

using OpenGL, to make my first 3D game prototypes. Over the following years I invested my entire freetime into this subject learning more and more about modern rendering techniques. I've learned tons, allowing me to contribute my ideas to invent new, bleeding-edge techniques.

2015

I created my first website

which taught me about modern web design. I kept expanding my knowledge with best practices like semantic elements, BEM and SASS.

2017

I made a social network

as a final exam. This opportunity helped me learn alot and gain experience with production database design, fontend- and backend web architecture.
